## Targeted Platforms

Mobile Application focus, targeting Android and Iphone IOS.

## Main Goals/Outcomes of the Application

The main goal of this application is to provide a more reliable, easy to use application for freelance, self employed or private business owners to track their earnings correctly and to be able to deduce how much tax they will need to pay. Think trades people, hair stylists, tattoo artists, small business owners such as pop up shops, cafes etc...

## Marketing strategy

This will be a SaaS start-up type application and business where the app is free to dowload but is limited to standard features unless the user purchases a premium subscription.

Standard Features (free application/no subscription)
- Add in daily hours
- Adjustable hourly wage
- see total income (net and gross) updated each day
- Adjustable tax rate
- Set pay period
- Able to edit and delete logged hours and change date of current hours being   logged
- track up to 3months of income and tax
- indicated MoM increase or decrease based off pay period

Premium Features (Locked behind a subscription)

- All of the above plus
- Track all monthly income and yearly review
- Able to get tax rundown for set period that you need to pay
- Able to export income and tax data to excel spreadsheet
- Dashboard analytics for overall finances with graphical displays of monthly earnings and tax, insights into where the most money is earned each pay period.

## Styling

Based off the current styling already in place, the overall feel should be a clean easy to use, professional SaaS start-up styling to the application. Mybe with some tasteful animations on the UI to make the application feel more interactive.

## Summary

This is a earnings tracker application aimed at small business owners, freelance workers and the self employed to easily help them track hours worked, earnings and tax. Run as SaaS model using an SQlite database where premium features are locked behind a subscription run through either the Android app store or the Apple app store depending on users device. Aimed solely at Mobile application for Android or iphone IOS.
